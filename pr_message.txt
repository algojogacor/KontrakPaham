🧪 Add tests for indonesianScore function

🎯 **What:**
Added unit tests for the `indonesianScore` function in `src/lib/documents.ts`. This addresses a testing gap where the core heuristic language detector lacked dedicated verification.

📊 **Coverage:**
The new test suite in `src/lib/documents.test.ts` covers the following scenarios:
* Empty strings and whitespace
* Typical high-scoring Indonesian contract text
* Prefix matches (`ber`, `mem`, etc.) correctly triggering `+0.5` hits
* Suffix matches (`kan`, `an`, `i`) correctly triggering `+0.4` hits
* Punctuation stripping logic to ensure trailing characters don't break word matching
* Pure non-Indonesian text resulting in a 0 score
* Case insensitivity
* Verifying the maximum score is capped at `1.0`

✨ **Result:**
Improved test coverage and reliability. The `indonesianScore` logic is now fully covered by tests, enabling safe refactoring in the future.
